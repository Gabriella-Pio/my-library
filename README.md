# MyLibrary

> Sistema de biblioteca com GCS aplicada — SENAI

## Tecnologias

- Backend: Spring Boot 3.x + Java 17 + H2 + JPA + Service Layer + DTOs
- Frontend: Angular 17 + Reactive Forms + Filtros + Dashboard

## Como Executar

### Backend

```
cd backend && mvn spring-boot:run
```

> Acesse: http://localhost:8080

### Frontend

```
cd frontend && npm install && ng serve
```

> Acesse: http://localhost:4200
